## Installation

### Download
```shell
$ curl -sSL https://raw.githubusercontent.com/saharben/cHWinfo/master/chwinfo.sh
```

### Make Executable
```shell
$ chmod +x chwinfo.sh
```

### Install
```shell
$ install chwinfo.sh /usr/local/bin/chwinfo | ln -s /usr/local/bin/chwinfo /usr/local/bin/etrshi
```

### Run
```shell
$ chwinfo
```
or
```shell
$ etrshi
```

### Enjoy
```shell
$ echo "Enjoy! :-)";
```
